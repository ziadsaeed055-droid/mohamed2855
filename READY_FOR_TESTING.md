## خطوات الاختبار الأساسية:

### 1. إضافة منتج
```
/dashboard/add-product → اختر الألوان → رفع صور → أضف المنتج
```
**المتوقع:** رسالة نجاح

### 2. عرض المنتج
```
/shop → ابحث عن المنتج → اختر لون → تحقق من الصورة
```
**المتوقع:** اسم اللون يظهر + الصورة تتغير

### 3. الفلترة
```
/shop → اختر لون من الفلاتر → تحقق من النتائج
```
**المتوقع:** فقط المنتجات ذات اللون تظهر
