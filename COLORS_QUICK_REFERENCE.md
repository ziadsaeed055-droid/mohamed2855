## 📌 الكيفية | How to Use

### إضافة منتج:
```
1. اذهب لصفحة "إضافة منتج"
2. في قسم الألوان، اضغط "إضافة لون"
3. ابحث عن اللون (مثلاً: "أزرق")
4. اختر درجة من الخيارات
5. اضغط "تأكيد"
6. البيانات المحفوظة:
   - colorId: "blue"
   - shadeId: "blue-500"
```

---

## 💾 البيانات المحفوظة في النظام | Saved Data Format

### صيغة اللون في قاعدة البيانات:
```typescript
{
  colorId: "blue",        // اسم اللون الرئيسي
  shadeId: "blue-500",    // معرّف الدرجة الكامل
  label: "أزرق",         // اسم اللون بالعربي
  hex: "#2196F3"          // الكود اللوني
}
```

## 🚀 نصائح مهمة | Important Tips

1. **اختر الدرجة المناسبة**:
   - للصيف: اختر الدرجات الفاتحة
   - للشتاء: اختر الدرجات الغامقة

2. **معاينة اللون**:
   - كل درجة عليها معاينة بصرية
   - شوف اللون قبل التأكيد

3. **أزرار التأكيد**:
   - أخضر = تأكيد
   - رمادي = إلغاء

4. **صور الألوان**:
   - رفع صورة خاصة لكل درجة لون
   - الصورة يجب أن تظهر اللون الصحيح

## ❓ الأسئلة الشائعة | FAQ

**س: كم درجة لون يمكن أن أختار لمنتج واحد؟**
ج: بدون حد أقصى، لكن التوصية 2-4 درجات

**س: هل يمكن تغيير اللون بعد الحفظ؟**
ج: نعم، افتح المنتج واضغط "تعديل الألوان"

**س: ما الفرق بين colorId و shadeId؟**
ج: 
- colorId = اللون الرئيسي (مثل "blue")
- shadeId = الدرجة المحددة (مثل "blue-500")

**س: كيف أعرف الكود اللوني (hex)?**
ج: يحفظ النظام تلقائياً مع اختيارك

**س: هل يدعم اللون الأسود والأبيض?**
ج: نعم، استخدم gray-50 للأبيض و gray-900 للأسود
