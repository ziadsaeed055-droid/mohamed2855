# اختبار الميزات الجديدة - Testing Guide ✅

## 🧪 خطوات الاختبار:

### صفحة تفاصيل المنتج (Product Details Page)

#### 1. Color Quick Bar Test
```
✓ انتقل لأي صفحة منتج
✓ ابحث عن شريط الألوان السريع (فوق الخيارات المتقدمة)
✓ اضغط على أي لون
✓ يجب أن تتغير الصورة الرئيسية مع تطبيق فلتر لون خفيف
✓ اسم اللون يجب أن يظهر في العنوان
```

#### 2. Color Preview Overlay Test
```
✓ اختر لوناً مختلفاً
✓ اراقب الصورة - يجب أن ترى فلتر لون خفيف ينطبق
✓ الفلتر يجب أن يكون:
  - قليل جداً للألوان الوسيطة
  - أكثر وضوحاً للألوان الداكنة جداً أو الفاتحة جداً
✓ إذا كانت هناك صورة محددة للون، لا يطبق الفلتر
```

#### 3. Stock Availability Test
```
✓ ابحث عن مؤشر التوفر تحت الألوان والمقاسات
✓ اختبر حالات مختلفة:
  - متوفر: تحقق من الرسالة الخضراء
  - محدود (3-5): تحقق من التحذير البرتقالي
  - غير متوفر: تحقق من الرسالة الحمراء مع زر "أخبرني"
```

### صفحة إضافة المنتج (Add Product Page)

#### 1. Color Quick Add Test
```
✓ افتح صفحة إضافة منتج
✓ اذهب إلى قسم "الألوان والمقاسات"
✓ اضغط على أي لون في شريط "اختر لوناً سريعاً"
✓ يجب أن يُضاف اللون تلقائياً للقائمة المختارة
✓ يظهر العدد في "الألوان المختارة"
```

#### 2. Advanced Search Test
```
✓ في نفس القسم، ابحث عن لون في "البحث والاختيار المتقدم"
✓ مثل: اكتب "أزرق" أو "blue"
✓ يجب أن تظهر جميع درجات الأزرق
✓ اختر درجة محددة (مثل "أزرق متوسط")
✓ اضغط على "تأكيد" أو زر موافقة
✓ يضاف للقائمة المختارة
```

#### 3. Color Images Upload Test
```
✓ بعد اختيار الألوان، انزل لقسم الصور
✓ لكل لون مختار يجب أن ترى:
  - مربع لون بالأعلى بالرقم الدقيق للون
  - اسم اللون
  - منطقة رفع الصور
✓ رفع صورة لكل لون
✓ الصورة تظهر مع زر حذف (×)
```

#### 4. Form Submission Test
```
✓ ملء باقي الحقول المطلوبة
✓ تأكد من اختيار:
  - اسم المنتج (عربي + إنجليزي)
  - الوصف
  - الأقسام
  - الأسعار
  - الكمية
  - الصورة الرئيسية
✓ ألوان وأسعار صحيحة معروضة
✓ اضغط "إرسال"
✓ يجب أن تُظهر رسالة نجاح
✓ تحويل للـ Dashboard
```

## 🚨 الحالات التي قد تظهر مشاكل:

### إذا كانت الصورة لا تتغير:
```
- تحقق: هل اللون المختار له صورة محددة في colorImages؟
- إذا نعم: يعرض الصورة مباشرة (صحيح)
- إذا لا: يعرض الصورة الأساسية مع فلتر (صحيح)
```

### إذا لم يظهر Stock Availability:
```
- تحقق: هل المنتج يحتوي على colorSizeStock data؟
- إذا لا: المكون يختفي تلقائياً (صحيح)
- إذا نعم: يجب أن يظهر الحالة
```

### إذا لم تظهر Color Quick Bar:
```
- تحقق: هل يوجد ألوان في المنتج؟
- إذا لا: المكون يختفي (صحيح)
- إذا نعم: يجب أن يظهر الشريط
```
