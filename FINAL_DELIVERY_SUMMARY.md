## 🔧 متطلبات Firebase Integration

### Collections المطلوبة:
```
firestore:
├── products/
│   └── colorSizeStock[] ← لتخزين المخزون
├── orders/
│   └── shippingUpdates[] ← لتتبع الطلبات
├── reviews/
│   └── ratings data ← للتقييمات
└── users/
    └── user data ← بيانات العملاء
```

### Firestore Indexes:
```
- orders (userId, status)
- reviews (productId, rating)
- products (category, price)
```

### Security Rules:
```
- Orders: read by owner or admin
- Reviews: read by all, write by authenticated
- Products: read by all, write by admin
```
