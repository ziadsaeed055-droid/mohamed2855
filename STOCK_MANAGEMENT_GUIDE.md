# دليل نظام إدارة المخزون

## نظرة عامة
نظام متكامل لإدارة المخزون والتحكم في كميات المنتجات حسب الألوان والمقاسات. يوفر التحقق من التوفر قبل الشراء وتنبيهات المخزون المحدود.

## المكونات الرئيسية

### 1. Stock Context (`contexts/stock-context.tsx`)
يوفر الدوال الأساسية للتعامل مع المخزون:
- `isStockAvailable()` - التحقق من توفر المخزون
- `getAvailableQuantity()` - الحصول على الكمية المتاحة
- `isLowStock()` - التحقق من المخزون المحدود
- `reserveStock()` - حجز المخزون مؤقتاً
- `confirmStockDeduction()` - تأكيد الخصم النهائي

### 2. Stock Utilities (`lib/stock-utils.ts`)
دوال مساعدة للعمل مع بيانات المخزون:
- `getColorSizeStock()` - الحصول على بيانات مخزون محددة
- `getStockStatus()` - الحصول على حالة المخزون (متوفر/محدود/غير متوفر)
- `getAvailableSizesForShade()` - المقاسات المتاحة لدرجة لون معينة
- `getTotalAvailableStock()` - إجمالي المخزون المتاح

### 3. مكونات واجهة المستخدم

#### StockStatusDisplay (`components/stock-status-display.tsx`)
عرض حالة المخزون في صفحة المنتج مع:
- شارة تحالف الحالة (أخضر/أصفر/أحمر)
- شريط تقدم نسبة التوفر
- رسائل تنبيهية

#### StockManagementDashboard (`components/stock-management-dashboard.tsx`)
لوحة تحكم في Dashboard لإدارة المخزون مع:
- عرض إجمالي المخزون
- عرض الكميات المخصصة للألوان والمقاسات
- تعديل وحذف إدخالات المخزون
- إحصائيات المخزون المحدود والمنتهي

## بنية البيانات

### ColorSizeStock Interface
```typescript
interface ColorSizeStock {
  shadeId: string          // معرف الدرجة: "blue-500"
  size: string            // المقاس: "M", "L"
  quantity: number        // الكمية الإجمالية
  reservedQuantity?: number  // الكمية المحجوزة
  isLowStock?: boolean   // هل المخزون محدود؟
}
```

## حالات الاستخدام

### 1. إضافة منتج جديد
1. ملء بيانات المنتج الأساسية
2. اختيار الألوان والدرجات
3. اختيار المقاسات
4. تعريف المخزون لكل مشترك (لون + مقاس)

### 2. التحقق من التوفر عند الشراء
```typescript
if (isStockAvailable(product, shadeId, size, quantity)) {
  addToCart(...)
} else {
  showError("المخزون غير كافي")
}
```

### 3. تحديث المخزون بعد الطلب
```typescript
confirmStockDeduction(productId, shadeId, size, quantity)
```

## الحالات والتنبيهات

| الحالة | اللون | الرسالة |
|-------|-------|--------|
| متوفر | أخضر | المنتج متاح |
| محدود | أصفر | المخزون محدود، تسرع |
| منتهي | أحمر | غير متوفر حالياً |

## التكامل مع الأنظمة الأخرى

### مع نظام السلة
يتحقق تلقائياً من المخزون قبل إضافة منتج للسلة

### مع نظام الطلبات
يقلل من المخزون عند تأكيد الطلب

### مع الإشعارات
يرسل إشعارات عند الوصول لمخزون محدود
